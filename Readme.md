## 功能说明

1. 获取音箱的日志
2. 获取到的日志存放在工程所在目录中应的日志文件夹

## 支持平台

- Windows
- Linux
- MacOS

## 依赖环境

- Python3
- adb

## 运行方法

- Windows: 双击 `main.cmd` 运行
- Linux/MaxOS: 进入工程所在目录，执行 `python3 main.py`
